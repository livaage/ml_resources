Curated Reading

Resources

External material I'd actually point students at — for the math, the code, and the deeper textbooks. Each entry has a one-line note on what it's good for.

Mathematics

The math foundations every ML practitioner needs. Picking favourites rather than trying to teach it here.

  • Mathematics for Machine Learning Free book by Deisenroth, Faisal, Ong. Covers exactly the math used in ML (linear algebra, vector calculus, probability, optimisation). Start here if you've forgotten the prerequisites.
  • 3Blue1Brown — Essence of Linear Algebra 15-video visual series. The best intuition pump for vectors, transformations, and eigendecomposition. Watch alongside any textbook.
  • 3Blue1Brown — Essence of Calculus Same author, same quality, for derivatives, integrals, and the chain rule. Essential before reading any backprop derivation.
  • Seeing Theory Interactive visual introduction to probability and statistics. Click through, don't read. Great for re-grounding intuition after a stats textbook.
  • MacKay — Information Theory, Inference, and Learning Free download. The classic on information theory + Bayesian inference. Dense but rewarding when you need entropy / KL / coding intuition.

Code & implementation

Where to read well-tested implementations and learn idiomatic patterns. The site won't reinvent these — they're already excellent.

  • scikit-learn user guide The gold standard for classical-ML reference code. Every algorithm has a worked example. Read the user guide, not just the API docs.
  • PyTorch tutorials Hands-on tutorials covering the framework most modern research uses. Start with the 60-minute blitz before going deeper.
  • Hugging Face docs The default entry point for working with pretrained transformers, datasets, and serving. transformers / datasets / accelerate are all essential reading.
  • Dive Into Deep Learning Free textbook with executable code for every concept (PyTorch / TF / JAX). The closest thing to a comprehensive DL textbook with runnable examples.
  • Papers With Code Papers indexed by task with linked implementations. Best place to find a strong baseline implementation for any benchmark.

Courses

Full curricula if you want a structured path rather than browsing topics.

  • Andrew Ng — Machine Learning Specialization The canonical "first ML course". Solid foundations, classical algorithms, light on deep learning. Pair with fast.ai for breadth.
  • fast.ai Top-down, hands-on deep learning. You build state-of-the-art models in week 1. Best complement to a theory-first course like Andrew Ng's.
  • CS231n — CNNs for Visual Recognition Stanford's deep-vision course. Slides and assignments are public. The gold standard for understanding CNNs and modern computer vision.
  • CS224n — NLP with Deep Learning Stanford's NLP course. Covers embeddings, attention, transformers. Lecture videos on YouTube; assignments are public.

Reference textbooks

When the TL;DRs aren't enough.

  • Goodfellow, Bengio, Courville — Deep Learning The classic. Free HTML version. Slightly dated on architectures (no transformers in detail) but the foundations chapters are still the standard reference.
  • Hastie, Tibshirani, Friedman — Elements of Statistical Learning The classical-ML reference. Free PDF. Read alongside ISLR for a friendlier introduction to the same material.
  • Murphy — Probabilistic Machine Learning Modern, comprehensive, free. Two volumes (foundations + advanced). Strongest treatment of probabilistic methods.
  • Sutton & Barto — Reinforcement Learning The standard RL textbook. Free PDF. Foundational for understanding policy gradients, value functions, and the modern derivatives.
